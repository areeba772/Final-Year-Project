Smart Cafe Management System (FYP)
An AI-powered, role-based cafe management solution designed for COMSATS University Islamabad, Vehari Campus. This system streamlines ordering, table reservations, and food authority monitoring.

🚀 Key Features
👤 User (Student/Teacher)
Smart Recommendations: Get personalized food suggestions powered by Scikit-learn based on your order history and top ratings.

Table Reservation: Book your dine-in spot by selecting a specific Time Slot (8 AM - 5 PM) and number of guests.

Real-time Orders: Browse the menu, add items to the cart, and place orders with JazzCash or COD options.

Feedback System: Rate and review food items to improve the AI's recommendations.

🛠️ Admin
Menu Management: Add, update, or delete food items with images and descriptions.

Order Tracking: Manage live order statuses (Pending, Cooking, Completed) and verify payments.

Analytics Dashboard: View total revenue, pending orders, and user statistics.

User Support: Reply to user messages and respond to Food Authority warnings.

⚖️ Food Authority
Price Monitoring: Inspect menu prices to ensure they meet university standards.

Quality Assurance: View customer reviews to monitor food quality and hygiene.

Warning System: Issue official notices to the Admin for any policy violations.

💻 Tech Stack
Frontend: HTML5, CSS3 (Custom Styles), JavaScript (Fetch API).
Backend: Python (Flask Framework).
Database: MySQL (Relational Database).
AI Engine: Scikit-learn (Recommendation System).
